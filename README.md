# RBWM 隐写页

一个用于 GitHub Pages 的浏览器端实验工具，演示在图片中嵌入与检测小段文本水印：

- **频域水印**：DCT / DWT 量化嵌入，兼顾抗压缩与抗转码。
- **空域伪随机模式**：基于种子随机分布 + 重复投票，对截图或缩放具备容错。
- **可组合**：通过复选框同时启用多个算法，提升鲁棒性。

## 使用方法

1. 打开 `index.html`（或部署到 GitHub Pages）。
2. 载入图片，填写要隐藏的短句，选择算法、随机种子与强度。
3. 点击“嵌入水印”生成结果，可下载 PNG。
4. 点击“提取/检测”从当前结果画布中提取水印，日志面板会给出各算法与融合结论。

> 工具面向教学与演示，不适合存放大文件或处理涉密内容。
