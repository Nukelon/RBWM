# RBWM 隐写页

一个可直接部署到 GitHub Pages 的前端工具页，支持 DCT / DWT 频域水印与空域伪随机扩频，可叠加算法，实现抗压缩、抗转码并在截图后仍能检测到的隐写文字。

## 功能特性
- 频域：8×8 DCT 与 Haar DWT 双通道水印，面向压缩、缩放等失真。
- 空域：伪随机扩频噪声写入，依赖密钥相关检测，截图后仍可检出。
- 16-bit 长度头 + UTF-8 编码文本，约 200 字符的短消息写入即可。
- 种子/强度可调，三种算法可通过复选框同时启用以叠加鲁棒性。
- 纯前端实现，手机/电脑均可直接加载使用。

## 使用步骤
1. 打开 `index.html`（或发布到 GitHub Pages 后访问）。
2. 选择原始图片，输入要隐藏的文字，设置密钥和强度。
3. 勾选要使用的算法，点击“写入水印”获取含水印图片并下载。
4. 在“检测/反操作”区域上传含水印的图片，使用相同密钥解析。

## 文件说明
- `index.html`：单页 UI、布局与样式。
- `main.js`：DCT/DWT/空域扩频的嵌入与提取逻辑，以及交互处理。

发布到 GitHub Pages 时，将仓库设置为 Pages 来源（`main` 分支根目录）即可。
